# btp_f_comp

前端组件

## startup

```bash
yarn
yarn gen # 进入交互式创建新组件
yarn dev # 平时的开发调试
```

打包发布

```bash
yarn build:lib # 打组件包
yarn build:doc # 打文档包
```
